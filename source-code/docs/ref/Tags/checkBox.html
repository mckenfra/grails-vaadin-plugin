A plugin for creating a Vaadin application in Grails.

(Quick Reference)

checkBox

Purpose

Configures a CheckBox field of the parent Vaadin Form component.

Examples

<v:form itemDataSource="${new BeanItem<Book>(bookInstance)}">
  <v:checkBox name="bestseller">Bestseller</v:checkBox>
</v:form>

Description

The checkBox tag is a special type of field tag, where the field component is a Vaadin CheckBox.

Body

  • Contains the caption for the field

Attributes

  • name (required) The property name of the field
  • caption (optional) The caption of the field. Overrides the body content.

See Vaadin CheckBox API docs for full list of allowed attributes.
