3.4 Views and Tags - Reference Documentation

Authors: Ondrej Kvasnovsky, Francis McKenzie

Version: 1.6.1-SNAPSHOT

3.4 Views and Tags

In the Vaadin view GSPs, you can use all of the Grails tags as you would in a normal Grails GSP.

Sitemesh layout tags are currently not supported. Please use the Vaadin layout tag instead.

In addition, you can embed Vaadin components in GSPs using the tag library provided by this plugin:

TagDescription
horizontalLayoutAdds a Vaadin HorizontalLayout component to the parent container.
verticalLayoutAdds a Vaadin VerticalLayout component to the parent container.
labelAdds a Vaadin Label component to the parent container.
layoutAdds a 'template-and-content' GspLayout to the parent container.
locationAdds a 'content-only' GspLayout component to the location in the parent container.
linkAdds a GrailsButton component to the parent container.
tableAdds a Vaadin Table component to the parent container.
columnConfigures a single column of the parent Vaadin Table component.
formAdds a Vaadin Form component to the parent container.
fieldConfigures a single field of the parent Vaadin Form component.
warningShows a warning-type Window.Notification in the application's main window.
errorShows an error-type Window.Notification in the application's main window.
