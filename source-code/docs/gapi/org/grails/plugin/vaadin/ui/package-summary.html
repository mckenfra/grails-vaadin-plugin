Package org.grails.plugin.vaadin.ui

A collection of UI components used in the scaffolded views.

 

Class Summary
BulletedList Mimics an HTML bulleted list (i.e. the '<ul>' tag)
DefaultButtons A button toolbar that provides the ability to add the same buttons that are provided in standard Grails scaffolding, i.e.: 'create', 'update', 'delete', 'edit'
DefaultCreate Mimics the Grails scaffolding create.gsp
DefaultEdit Mimics the Grails scaffolding edit.gsp
DefaultHome Mimics the Grails root index.gsp (i.e. showing Grails installation details, controllers etc.).
DefaultList Mimics the Grails scaffolding list.gsp
DefaultLogo Displays the grails header logo.
DefaultNavigation A navigation toolbar that provides the ability to add the same buttons that are provided in standard Grails scaffolding, i.e.: 'home', 'create', 'list'
DefaultShow Mimics the Grails scaffolding show.gsp
DefaultWindow Mimics the Grails main.gsp top-level template.
GrailsButton Mimics the g:link tag
